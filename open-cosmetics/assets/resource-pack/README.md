# Resource Pack Open Cosmetics

Snapshot degli asset cosmetici arma per Open Cosmetics.

Questo pack e' pensato per essere caricato insieme al resource pack di
Open Weapons. Open Cosmetics fornisce modelli, texture e suoni cosmetici; Open
Weapons mantiene i dispatcher item e le varianti custom model data delle armi.

Contenuto principale:

- modelli LED, colorabili e skin;
- texture delle finiture cosmetiche;
- modelli dei gettoni LED/colore;
- suoni skin `sugarline` e `royal`.

Per generare uno zip pronto per Minecraft dalla radice della repository:

```bash
bash scripts/build-resource-packs.sh
```
