# Asset Open Cosmetics

Questa cartella contiene gli asset pubblicabili del modulo `open-cosmetics`.

## Resource pack

Il sotto-pack `resource-pack/` include modelli, texture e suoni per:

- LED arma;
- varianti colorabili;
- skin cosmetiche;
- gettoni cosmetici;
- suoni custom delle skin.

I dispatcher item e le tabelle custom model data delle armi restano nel
resource pack di `open-weapons`, perche' Open Weapons rimane proprietario delle
varianti visuali definite in `weapons.yml`.
